# Maqueteo Pro

App para organizar y planificar proyectos de maquetacion artesanal.

Proyecto amateur hecho con entusiasmo por **Artesana Dominguera**, parte del ecosistema [Encuaderna Apps](https://encuaderna.github.io/encuaderna-apps/).

## Usar la aplicacion

Entra directamente desde tu navegador, sin instalar nada:

**[Abrir Maqueteo Pro](https://amber-bind-your-book.base44.app)**

## Que encontraras

- Planificacion de etapas de trabajo
- Seguimiento de proyectos de maquetacion
- Herramientas simples para principiantes
- Acceso directo desde el navegador

## Sobre este proyecto

Maqueteo Pro es parte de Encuaderna Apps, un conjunto de herramientas sencillas para dar los primeros pasos en encuadernacion y otras manualidades.

[Ver todas las apps](https://encuaderna.github.io/encuaderna-apps/)
